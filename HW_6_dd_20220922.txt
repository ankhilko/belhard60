1. Написать функцию перевода десятичного числа в двоичное и обратно, без
использования функции int

2. Код Морзе для представления цифр и букв использует тире и точки. Напишите
функцию для кодирования текстового сообщения в соответствии с кодом Морзе.
(словари в помощь)

3. Дан список чисел и на вход поступает число N, необходимо сместить список на
указанное число, пример: [1,2,3,4,5,6,7] N=3 ответ: [5,6,7,1,2,3,4]

4. Дан список содержащий в себе различные типы данных, отфильтровать таким
образом, чтобы остались только строки, использование дополнительного списка
незаконно

5. Дан список чисел, необходимо его развернуть без использования метода revese и
функции reversed, а так же дополнительного списка и среза

6. Дан список рандомных чисел, необходимо отсортировать его в виде, сначала
четные, потом нечётные

7. Дан список чисел, необходимо для каждого элемента посчитать сумму его
соседей, для крайних чисел одним из соседей является число с противоположной
стороны списка

8. Дан словарь, ключ - Название страны, значение - список городов, на вход
поступает город, необходимо сказать из какой он страны

9. Дан словарь словарей, ключ внешнего словаря - id пользователя, значение -
словарь с данными о пользователе (имя, фамилия, телефон, почта), вывести
имена тех, у кого не указана почта (нет ключа email или значение этого ключа -
пустая строка)
